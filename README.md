# rocketWebUI

Web UI for Rocket

Uses Python and JavaScript to procces data and chart in real time.

## How to Use

Start Python webserver:
```
python3 -m http.server:3000
```

Start Python script:
```
python3 websocket_server.py
```

## Libraries and other resources used

### Python
- tornado

### JavaScript
- d3.js
- dc.js
- crossfilter.js
